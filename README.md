### Open VTuber JS

> In progress

This project is for people to become VTuber by opening a website without installing any softwares.

<img src="./.github/demo/1.png" alt="1" style="width:100%;" />

##### Dev Installation

```
yarn install
yarn start
```

Then open `localhost:3000`
